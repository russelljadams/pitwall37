# PitWall37 Racing Engineering Library

## How to Use
This library contains racing engineering reference materials for the PitWall37 race agent.
The race engineer can read these files to provide data-backed answers about vehicle dynamics,
aerodynamics, suspension setup, damper tuning, tire behavior, and data analysis.

PDFs can be read with the Read tool (for text-based PDFs). Markdown files contain
pre-extracted web content ready for immediate reference.

**Total files:** 33
**Total size:** ~69 MB
